---- Important Tips ----

1. Download the project zip file
2. Extract the file and copy 'chat' folder
3.Paste inside root directory(for xampp xampp/htdocs)
4.Open PHPMyAdmin (http://localhost/phpmyadmin)
5. Create a database with the name 'chat'
6. Import chat.sql file (given inside the zip package in SQL file folder)
7. Run the script http://localhost/chat
